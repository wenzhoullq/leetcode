## 题目

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组，求出这个数组中的逆序对的总数。

## 思路

用并归的思想，在后面那块移动的时候就知道前面有大于它的部分了

```java
class Solution {
    int[] nums,temp;
    public int reversePairs(int[] nums) {
        temp=new int[nums.length];
        this.nums=nums;
        return guibing(0,nums.length-1);
    }
   private int guibing(int left,int right){
        if(left>=right) return 0;
        int mid=(left+right)/2;
        int ans=guibing(left,mid)+guibing(mid+1,right);
        for(int i=0;i<nums.length;i++) temp[i]=nums[i];
        int i=left,j=mid+1;
        for(int k=left;k<=right;k++){
            if(i==mid+1) nums[k]=temp[j++];
            else if(j==right+1||temp[i]<=temp[j]) nums[k]=temp[i++];
            else {
                nums[k]=temp[j++];
                ans+=mid-i+1;
            }
        }
        return ans;
    }
}
```
