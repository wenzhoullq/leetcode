## 题目
输入两个链表，找出它们的第一个公共节点。
## 思路
连接消除`长度差`
```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p1=headA,p2=headB;
        while(p1!=p2){
            if(p1!=null) p1=p1.next;
            else p1=headB;
            if(p2!=null) p2=p2.next;
            else p2=headA;
        }
        return p1;
    }
}
```
