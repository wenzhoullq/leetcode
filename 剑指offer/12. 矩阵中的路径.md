## 题目
给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。
## 思路
经典回溯题
```java
class Solution {
    boolean flag=false;
    String word;
    int[][] direct={{1,0},{0,1},{-1,0},{0,-1}},visit;
    int xlength,ylength;
    char[] temp;
    void huisu(int x,int y,int step,char[][] board){
        if(step==word.length()){
            flag=true;
            return ;
        }
        if(x<0||x==xlength||y<0||y==ylength||visit[y][x]==1||word.charAt(step)!=board[y][x]) return ;
        visit[y][x]=1;
        temp[step]=board[y][x];
        for(int i=0;i<4;i++){
            huisu(x+direct[i][0],y+direct[i][1],step+1,board);
            if(flag) return ;
        }
        visit[y][x]=0;
    }
    public boolean exist(char[][] board, String w) {
        if(board.length==0||board[0].length==0) return false;
        temp=new char[w.length()];
        xlength=board[0].length;ylength=board.length;
        visit=new int[ylength][xlength];
        word=w;
        for(int y=0;y<ylength;y++){
            for(int x=0;x<xlength;x++){
                huisu(x,y,0,board);
                if(flag) return true;
            }
        }
        return false;
    }
}
```
