## 题目
给定单向链表的头指针和一个要删除的节点的值，定义一个函数删除该节点。

返回删除后的链表的头节点。
## 思路
双指针
```java
class Solution {
    public ListNode deleteNode(ListNode head, int val) {
        ListNode pre=null,cur=head;
        while(cur.val!=val){
            pre=cur;
            cur=cur.next;
        }
        if(pre!=null) pre.next=cur.next;
        else head=head.next;
        return head;
    }
}
```
