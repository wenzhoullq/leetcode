## 题目
数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。
## 思路
摩尔投票法，算众数用的
```java
class Solution {
    public int majorityElement(int[] nums) {
        int ans=nums[0],count=1;
        for(int i=1;i<nums.length;i++){
            if(ans!=nums[i]) count--;
            else count++;
            if(count<0) {
                ans=nums[i];count=1;
            }
        }
        return ans;
    }
}
```
