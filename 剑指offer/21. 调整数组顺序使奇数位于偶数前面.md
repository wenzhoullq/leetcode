## 题目
输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有奇数在数组的前半部分，所有偶数在数组的后半部分。
## 思路
双指针
```java
class Solution {
    void swap(int[] nums,int start,int end){
        int temp=nums[start];
        nums[start]=nums[end];
        nums[end]=temp;
    }
    public int[] exchange(int[] nums) {
        int start=0,end=nums.length-1;
        while(start<end){
            if(nums[start]%2==0) swap(nums,start,end--);
            else start++;
        }
        return nums;
    }
}
```
