## 题目
给定一棵二叉搜索树，请找出其中第k大的节点。
## 思路
中序遍历逆序即可
```java
class Solution {
    boolean flag=false;
    int k,ans=0;
    void dfs(TreeNode root){
        if(flag||root==null) return ;
        dfs(root.right);
        if(--k==0) {
            flag=true;
            ans=root.val;
            return ;
        }
        dfs(root.left);
    }
    public int kthLargest(TreeNode root, int k) {
        this.k=k;
        dfs(root);
        return ans;
    }
}
```
