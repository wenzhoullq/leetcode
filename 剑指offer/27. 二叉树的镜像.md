## 题目
请完成一个函数，输入一个二叉树，该函数输出它的镜像。
## 思路
经典递归题
```java
class Solution {
    public TreeNode mirrorTree(TreeNode root) {
        if(root==null) return null;
        mirrorTree(root.left);
        mirrorTree(root.right);
        TreeNode temp;
        temp=root.left;
        root.left=root.right;
        root.right=temp;
        return root;
    }
}
```
