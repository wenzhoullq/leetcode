## 题目
写一个函数，输入 n ，求斐波那契（Fibonacci）数列的第 n 项（即 F(N)）。斐波那契数列的定义如下：

F(0) = 0,   F(1) = 1
F(N) = F(N - 1) + F(N - 2), 其中 N > 1.

## 思路
动态规划简单题
```java
class Solution {
    public int fib(int n) {
        if(n==0) return 0;
        if(n==1) return 1;
        int[] dp=new int[n+1];
        dp[1]=1;
        for(int i=2;i<=n;i++) dp[i]=(dp[i-1]+dp[i-2])%1000000007;
        return dp[n];
    }
}
```
