## 题目

假设把某股票的价格按照时间先后顺序存储在数组中，请问买卖该股票一次可能获得的最大利润是多少？

## 思路

简单题，没啥思路

```java
class Solution {
    public int maxProfit(int[] prices) {
        int min=Integer.MAX_VALUE,pro=0;
        for(int i=0;i<prices.length;i++){
            min=Math.min(min,prices[i]);
            pro=Math.max(prices[i]-min,pro);
        }
        return pro;
    }
}

```
