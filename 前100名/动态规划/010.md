## 思路
填表，具体的细节再代码处
```java
class Solution {
    public boolean isMatch(String s, String p) {
        boolean[][] dp=new boolean[s.length()+1][p.length()+1];
        dp[0][0]=true;
        for(int i=0;i<=s.length();i++){
            char c1=' ';
            if(i!=0) c1=s.charAt(i-1);
            for(int j=1;j<=p.length();j++){
                char c2=p.charAt(j-1);
                // System.out.println(i+" "+c2);
                if(i==0){
                    if(c2=='*')dp[i][j]=dp[i][j-2];
                } 
                else{
                    if(c1==c2||c2=='.') dp[i][j]=dp[i-1][j-1];
                    else if(c2=='*'){
                        dp[i][j]=dp[i][j-2];//抵消0次
                        char c3=p.charAt(j-2);
                        if(c3==c1||c3=='.') {
                            dp[i][j]=dp[i][j]||dp[i-1][j];//抵消两次以上，抵消一次的已经在上一回合做过，可以不写
                        }
                    }
                }    
            }
        }
        return dp[s.length()][p.length()];
    }
}
```
