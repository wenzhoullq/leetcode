## 题目
一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

## 思路

有多少路径，经典的`动态规划`题

```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[][] dp=new int[m][n];
        for(int y=0;y<m;y++){
            for(int x=0;x<n;x++){
                if(x==0||y==0){
                    dp[y][x]=1;
                    continue;
                }
                dp[y][x]=dp[y-1][x]+dp[y][x-1];
            }
        }
        return dp[m-1][n-1];
    }
}
```
