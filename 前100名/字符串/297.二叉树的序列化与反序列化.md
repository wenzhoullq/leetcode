## 题目

序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。

请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

## 思路

这里记录的形式都是层次遍历的结果，因此无论是序列化还是反序列化都需要层次遍历

```java
public class Codec {
    // Encodes a tree to a single string.
    public String serialize(TreeNode root) {
        if(root==null) return "[]";
        StringBuilder ans=new StringBuilder("[");
        Queue<TreeNode> q=new LinkedList<TreeNode>();
        q.add(root);
        while(!q.isEmpty()){
            TreeNode temp=q.poll();
            if(temp!=null){
                ans.append(temp.val+",");
                q.add(temp.left);
                q.add(temp.right);
            }
            else ans.append("null,");
        }
        ans.deleteCharAt(ans.length()-1);//去除多余的 ，方法是deleteCharAt
        return ans.append("]").toString();
    }

    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) {
        if(data.equals("[]")) return null;
        String[] temps=data.substring(1,data.length()-1).split(",");//去除[ ] 记住分类方法拼写
        int count=0;
        TreeNode root=new TreeNode(Integer.parseInt(temps[count++]));
        Queue<TreeNode> q=new LinkedList<TreeNode>();
        q.add(root);
        while(!q.isEmpty()&&count<temps.length){//需要注意的是，这里是对String数组的加入，为null的话不可能会进入Q队列，只需循环判定即可
            TreeNode temp=q.poll();
            if(!temps[count].equals("null")) {
                temp.left=new TreeNode(Integer.parseInt(temps[count]));
                q.add(temp.left);
            }
            count++;
            if(!temps[count].equals("null")) {
                temp.right=new TreeNode(Integer.parseInt(temps[count]));
                q.add(temp.right);
            }
            count++;
        }
        return root;
    }
}
```
