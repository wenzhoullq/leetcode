## 题目
编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。
## 思路
公共前缀是从0开始的，所以固定一个对比即可
```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
        if(strs.length==0) return "";
        String ans=strs[0];
        for(int i=1;i<strs.length;i++){
            int length=0;
            String temp=strs[i];
            while(length<ans.length()&&length<temp.length()){
                if(ans.charAt(length)!=temp.charAt(length)) break;
                length++;
            }
            if(ans.substring(0,length).equals("")) return "";
            ans=ans.substring(0,length);
        }
        return ans;
    }
}
```
