## 题目

给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。
单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

## 思路
经典回溯题，方向数组和visit数组

```java
class Solution {
    char[] w;
    boolean flag=false;
    int[][] visit,direct={{0,1},{0,-1},{-1,0},{1,0}};
    int length;
    void huisu(int y,int x,int index, char[] temp,char[][] board){
        if(flag) return;
        if(index==length){
            if(Arrays.equals(w,temp)) flag=true;//比较两个数组是否相同，用Arrays里的方法
            return ;
        } 
        if(x<0||x==board[0].length||y<0||y==board.length||visit[y][x]==1) return ;
        visit[y][x]=1;
        temp[index]=board[y][x];
        for(int i=0;i<4;i++){
            huisu(y+direct[i][0],x+direct[i][1],index+1,temp,board);
        }
        visit[y][x]=0;
    }
    public boolean exist(char[][] board, String word) {
        if(board.length==0||board[0].length==0||word.length()==0) return flag;
        w=word.toCharArray();//注意方法的大小写
        length=word.length();
        visit=new int[board.length][board[0].length];
        char[] temp=new char[length];
        for(int y=0;y<board.length;y++){
            for(int x=0;x<board[0].length;x++){
                huisu(y,x,0,temp,board);
                if(flag) return true;
            }
        }
        return flag;
    }
}
```
