## 题目

给定一个不含重复数字的数组 nums ，返回其 所有可能的全排列 。你可以 按任意顺序 返回答案。

## 思路

因为返回的是`所有可能性`，用回溯，用动态规划十分的麻烦，有炫技的嫌疑。

```java
class Solution {
    List<List<Integer>> ans=new LinkedList<>();
    int[] num;
    HashMap<Integer,Integer> m=new HashMap<>();
    void huisu(int n,LinkedList<Integer> temp){
        if(n==num.length){
            ans.add(new LinkedList<Integer>(temp));
            return ;
        }
        for(int i=0;i<num.length;i++){  // 因为这里是从0开始，所以将添加部分在for循环内实现
            if(m.containsKey(num[i])) continue;
            m.put(num[i],1);
            temp.add(num[i]);
            huisu(n+1,temp);
            temp.removeLast();
            m.remove(num[i]);
        }
    };
    public List<List<Integer>> permute(int[] nums) {
        num=nums;
        LinkedList<Integer> temp=new LinkedList<>();
        huisu(0,temp);
        return ans;
    }
}

```
