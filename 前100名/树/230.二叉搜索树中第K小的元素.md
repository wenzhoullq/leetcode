### 题目

给定一个二叉搜索树的根节点 `root` ，和一个整数 `k` ，请你设计一个算法查找其中第 `k` 个最小元素（从 1 开始计数）。

###  思路

中序遍历

```java
class Solution {
    int count=0,ans=0,k;
    boolean flag=false;
    void dfs(TreeNode root){
        if(root==null||flag) return ;
        dfs(root.left);
        if(++count==k) {
            ans=root.val;
            flag=true;
            return ;
        }
        dfs(root.right);
    }
    public int kthSmallest(TreeNode root, int k) {
        this.k=k;
        dfs(root);
        return ans;
    }
}
```
