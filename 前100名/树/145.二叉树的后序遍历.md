### 题目

给定一个二叉树，返回它的 *后序* 遍历。

### 思路

先写递归，迭代下次再写

```java
class Solution {
    List<Integer> ans=new ArrayList<>();
    void dfs(TreeNode root){
        if(root==null) return ;
        dfs(root.left);
        dfs(root.right);
        ans.add(root.val);
    }
    public List<Integer> postorderTraversal(TreeNode root) {
        dfs(root);
        return ans;
    }
}
```
