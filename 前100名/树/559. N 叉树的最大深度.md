## 题目
给定一个 N 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

N 叉树输入按层序遍历序列化表示，每组子节点由空值分隔（请参见示例）。
## 思路
深度优先
```java
class Solution {
    int dfs(Node root){
        if(root==null) return 0;
        int deep=1;
        for(int i=0;i<root.children.size();i++){
            deep=Math.max(deep,dfs(root.children.get(i))+1);
        }
        return deep;
    }
    public int maxDepth(Node root) {
        return dfs(root);
    }
}
```
