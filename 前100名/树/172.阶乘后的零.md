## 题目
给定一个整数 n ，返回 n! 结果中尾随零的数量。

提示 n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1
## 思路
0肯定是由质因子2 * 5 得来的，因此统计5的个数即可，n/5就是带5质因子的数字的数目，但是n/5后的另一个质因子可能也带5的质因子，因此继续n/5
```java
class Solution {
    public int trailingZeroes(int n) {
        int count = 0;
        while (n > 0) {
        n /= 5;
        count += n;
        }
        return count;
    }
}
```
