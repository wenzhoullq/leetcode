### 题目

给你一个以字符串表示的非负整数 `num` 和一个整数 `k` ，移除这个数中的 `k` 位数字，使得剩下的数字最小。请你以字符串形式返回这个最小的数字。

### 思路

双向链表的单调栈，碰到比它小的就弹出

```java
class Solution {
    public String removeKdigits(String num, int k) {
        Deque<Integer> q=new LinkedList<Integer>();
        int[] temp=new int[num.length()+2];
        for(int i=0;i<num.length();i++) temp[i+1]=num.charAt(i)-'0';
        int start=0,count=0;
        StringBuilder ans=new StringBuilder();
        boolean hasstart=true;
        while(start<temp.length){
            if(count==k) {
                q.add(start++);
                continue;
            }
            while(!q.isEmpty()&&temp[q.getLast()]>temp[start]&&count<k) {
                q.removeLast();
                count++;
            }
            q.add(start++);
        }
        while(!q.isEmpty()){
            char tempchar=(char)(temp[q.getFirst()]+'0');
            if(tempchar=='0'&&hasstart){
                q.removeFirst();continue;
            }
            hasstart=false;
            ans.append(tempchar);
            q.removeFirst();
        }
        return !"".equals(ans.toString())?ans.toString().substring(0,ans.length()-1):"0";
    }
}
```
