## 题目
给你一个非负整数 x ，计算并返回 x 的 算术平方根 。
由于返回类型是整数，结果只保留 整数部分 ，小数部分将被 舍去 。
## 思路
牛顿迭代法，先画出切线，然后不断逼近
```java
class Solution {
    int x;
    public int mySqrt(int x) {
        this.x=x;
        if(x==0) return 0;
        return (int)sqrt(x);
    }
    double sqrt(double res){
       double temp=(res+x/res)/2;
       if(temp==res) return res;
       else return sqrt(temp);
    }
}
```
