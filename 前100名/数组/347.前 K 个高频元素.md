## 题目

给你一个整数数组 `nums` 和一个整数 `k` ，请你返回其中出现频率前 `k` 高的元素。你可以按 **任意顺序** 返回答案。

## 思路

思路一是大根堆，思路二是桶排

* 大根堆

  ```java
  class Solution {
      public int[] topKFrequent(int[] nums, int k) {
          HashMap<Integer,Integer> m=new HashMap<>();
          for(int num:nums) m.put(num,m.getOrDefault(num,0)+1);
          Queue<Integer> q=new PriorityQueue<>((x,y)->m.get(y)-m.get(x));//比较的部分是m.get()
          for(int n:m.keySet()) q.add(n);
          int[] ans=new int[k];
          for(int i=0;i<k;i++) ans[i]=q.poll();
          return ans;
      }
  }
  ```

  

* 桶排

  ```java
  class Solution {
      public int[] topKFrequent(int[] nums, int k) {
          HashMap<Integer,Integer> m=new HashMap<>();
          for(int num:nums) m.put(num,m.getOrDefault(num,0)+1);
          List<Integer>[] temp=new ArrayList[nums.length+1];
          int[] ans=new int[k];
          for(int i:m.keySet()){
              if(temp[m.get(i)]==null) temp[m.get(i)]=new ArrayList<Integer>();
              temp[m.get(i)].add(i);
          }
          for(int i=temp.length-1,count=0;i>=0&&count<k;i--){
              if(temp[i]!=null) {
                  for(int j=0;j<temp[i].size()&&count<k;j++) ans[count++]=temp[i].get(j);
              }
          }
          return ans;
      }
  }
  ```
