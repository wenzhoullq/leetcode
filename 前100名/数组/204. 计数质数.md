## 题目
给定整数 n ，返回 所有小于非负整数 n 的质数的数量 。
## 思路
埃式筛选 找到一个质数，对它的倍数不断进行标价非质数
```java
class Solution {
    public int countPrimes(int n) {
        boolean[] notPrimes = new boolean[n];
        int count = 0;
        for (int i = 2; i < n; i++) {
            if (notPrimes[i])
                continue;
            count++;
            for (int j = i; j < n; j += i)
                notPrimes[j] = true;
        }
        return count;
    }
}
```
