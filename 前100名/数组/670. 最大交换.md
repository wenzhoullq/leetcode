## 题目
给定一个非负整数，你至多可以交换一次数字中的任意两位。返回你能得到的最大值。
## 思路
双指针，一个头部指针，一个尾部指针
```java
class Solution {
    public int maximumSwap(int num) {
        char[] ans=Integer.toString(num).toCharArray();
        for(int i=0;i<ans.length;i++){
            int index=-1;
            for(int j=ans.length-1;j>i;j--){
                if(ans[i]<ans[j]){
                   if(index==-1) index=j;
                   else if(ans[index]<ans[j]){
                       index=j;
                   }
                }
            }
            if(index!=-1){
                char temp=ans[index];
                ans[index]=ans[i];
                ans[i]=temp;
                return Integer.parseInt(new String(ans));
            }
        }
        return num;
    }
}
```
