## 题目

给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。
## 思路
最笨的思路就是，对每一个数字都计算出来有多少个1，其次比较巧妙地想法就是如果是奇数，那么它比它前一个数的1多一个，如果是偶数，那么它跟它的1/2的个数是一样多的

```java
class Solution {
    public int[] countBits(int n) {
        int[] ans=new int[n+1];
        for(int i=1;i<=n;i++){
            if(i%2==1) ans[i]=ans[i-1]+1;
            else ans[i]=ans[i/2];
        }
        return ans;
    }
}
```
