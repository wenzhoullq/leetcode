### 题目

给定一个包含非负整数的数组，你的任务是统计其中可以组成三角形三条边的三元组个数。

### 思路

双指针+二分查找

```java
class Solution {
    public int triangleNumber(int[] nums) {
        Arrays.sort(nums);
        int ans=0;
        for(int i=0;i<nums.length-2;i++){
            if(nums[i]==0) continue;
            for(int j=i+1;j<nums.length-1;j++){
                int left=j+1,right=nums.length-1,target=nums[i]+nums[j],k=j;
                while(left<=right){
                    int mid=(left+right)/2;
                    if(target>nums[mid]) {
                        left=mid+1;
                        k=mid;
                    }
                    else right=mid-1;
                }
                ans=ans+k-j;
            }
        }
        return ans;
    }
}
```
