### 题目

给定一个正整数 *n*，将其拆分为**至少**两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

### 思路

剪绳子题

```java
class Solution {
    public int integerBreak(int n) {
        if(n==2) return 1;
        if(n==3) return 2;
        if(n==4) return 4;
        int ans=1;
        while(n>4){
            ans*=3;
            n-=3;
        }
        return ans*n;
    }
}

```
