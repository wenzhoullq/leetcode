## 题目
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
## 思路
一个很巧妙的方法，遇到非0数字，按顺序排列，最后剩下的全部补零即可
```java
class Solution {
    public void moveZeroes(int[] nums) {
        int start=0;
        for(int i=0;i<nums.length;i++){
            if(nums[i]!=0)  nums[start++]=nums[i];
        }
        for(int i=start;i<nums.length;i++){
            nums[i]=0;
        }
    }
}
```
