### 题目

给定一个包含 `[0, n]` 中 `n` 个数的数组 `nums` ，找出 `[0, n]` 这个范围内没有出现在数组中的那个数。

### 思路

原地哈希/新建数组

```java
class Solution {
    public int missingNumber(int[] nums) {
        boolean lostend=true;
        int i=0;
        while(i<nums.length){
            if(nums[i]==nums.length) {
                i++;
                lostend=false;
                continue;
            }
            if(i!=nums[i]){
                int temp=nums[i];
                nums[i]=nums[temp];
                nums[temp]=temp;
            }
            else {
                i++;
                continue;
            }
        }
        if(lostend) return nums.length;
        for(i=0;i<nums.length;i++) {
            if(i!=nums[i]) return i;
        }
        return 0;
    }
}
```
