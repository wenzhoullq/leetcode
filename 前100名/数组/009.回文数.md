### 题目

给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。例如，121 是回文，而 123 不是。

### 思路

数字倒着计算看是否满足回文

```java
class Solution {
    public boolean isPalindrome(int x) {
        if(x<0) return false;
        int temp=x,count=0;
        while(temp>0){
            count=count*10+temp%10;
            temp/=10;
        }
        return count==x;
    }
}
```
