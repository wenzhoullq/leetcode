### 题目

给定一个头结点为 `head` 的非空单链表，返回链表的中间结点。

如果有两个中间结点，则返回第二个中间结点。

### 思路

快慢指针

```java
class Solution {
    public ListNode middleNode(ListNode head) {
        ListNode slow=head,quick=head;
        while(quick!=null&&quick.next!=null){
            slow=slow.next;
            quick=quick.next.next;
        }
        return slow;
    }
}
```
