## 题目

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

* 优先队列
```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        PriorityQueue<ListNode> q=new PriorityQueue<>((x,y)->x.val-y.val);
        for(int i=0;i<lists.length;i++){
           if(lists[i]!=null) q.offer(lists[i]);
        }
        ListNode dummy=new ListNode(),tail=dummy;
        while(!q.isEmpty()){
            ListNode temp=q.poll();
            tail.next=temp;
            tail=tail.next;
            if(temp.next!=null) q.offer(temp.next);
        }
        return dummy.next;
    }
}
```

* 归并

```java
class Solution {
    ListNode ans;
    ListNode[] lists;
    ListNode merge(ListNode l1,ListNode l2){
        if(l1==null) return l2;
        if(l2==null) return l1;
        if(l1.val<l2.val){
            l1.next=merge(l1.next,l2);
            return l1;
        }
        else {
            l2.next=merge(l2.next,l1);
            return l2;
        }
    }
    ListNode binggui(int left,int right){
        if(left==right) return lists[left];
        int mid=(left+right)/2;
        ListNode leftList=binggui(left,mid);
        ListNode rightList=binggui(mid+1,right);
       return  merge(leftList,rightList);
    }
    public ListNode mergeKLists(ListNode[] lists) {
        if(lists==null||lists.length==0) return null;
        this.lists=lists;
       return binggui(0,lists.length-1);
    }
}
```
