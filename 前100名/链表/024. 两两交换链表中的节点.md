## 题目
给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。
## 思路
设置两条链表分别记录就可以了
```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        ListNode l1=new ListNode(0),l2=new ListNode(1),dummy=new ListNode(0),p=dummy,L1=l1,L2=l2;
        int count=0;
        while(head!=null){
            ListNode temp=head.next;
            head.next=null;
            if(count++%2==0) {
                L1.next=head;
                L1=L1.next;
            }
            else {
                L2.next=head;
                L2=L2.next;
            }
            head=temp;
        }
        l1=l1.next;
        l2=l2.next;
        while(l1!=null||l2!=null){
            if(l2!=null){
                ListNode temp=l2.next;
                l2.next=null;
                p.next=l2;
                l2=temp;
                p=p.next;
            }
            if(l1!=null) {
                ListNode temp=l1.next;
                l1.next=null;
                p.next=l1;
                l1=temp;
                p=p.next;
            }
        }
        return dummy.next;
    }
}
```
