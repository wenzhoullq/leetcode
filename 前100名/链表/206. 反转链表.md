## 题目
给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。
## 思路 
一开始用栈，后面用双指针。
```java
class Solution {
    public ListNode reverseList(ListNode head) {
            ListNode pre=null,cur=head,temp;
            while(cur!=null){
                temp=cur.next;
                cur.next=pre;
                pre=cur;
                cur=temp;
            }
            return pre;
    }
}
```
